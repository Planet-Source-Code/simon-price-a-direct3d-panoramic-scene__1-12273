CylinderView v1.0

by Simon Price

(pictures by Craig Nisbet)

What's this?

CylinderView is a program that takes a .jpg or .bmp image and wraps it onto a sphere using DirectX. It then allows you to rotate the camera around the scene. It fools you into thinking that a whole massive 3D scene is being rendered! Because it's really just a cylinder, it can run a phenomenal speed - over 60 FPS on my PC!

What features has it got?

  • Uses DirectX7 - so it must be installed!

  • Loads .jpg and .bmp images

  • Supports hardware and software rendering

  • Variable polygon count - you choose the rendering quality/speed

  • Variable picture resolution - you choose the resolution of the the graphics

How do I use it?

It's pretty self explanatory, just choose the options and click OK. Then use the mouse to rotate the camera around. If it displays an error message, it will end the program, but you can restart and select different options to avoid the error by selecting software rendering of 256 x 256 pixel textures. Press escape to end the program.

I want more pictures!

If you want more pictures, you can make them yourself, or I could ask Craig to make some more, and once you have them you need to put them in the \data\maps directory for the program to find them. Make sure your pictures have sizes that are a power of 2 - e.g. 128, 256, 512, 1024 etc. Also understand that some 3D cards only support square textures or ones up to a certain size.
